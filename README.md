# Deformation-Lamps
変幻灯，光のパタンを投影することで，静止画像にリアルな動きの印象を与えることのできるまったく新しい発想に基づく光投影技術です．人間の錯覚を巧みに利用した「変幻灯」は，止まっているはずのものが動いて見えるという，かつてなかった視覚体験を生み出します．

言語：Python 3.8.1

ライブラリ：OpenCV, numpy
